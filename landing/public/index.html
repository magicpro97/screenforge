Quick Start

1

Install globally with npm:

npm install -g screenforge
2

Generate all icon sizes from a single image:

screenforge icon my-icon.png --platform all
3

Create splash screens for all devices:

screenforge splash logo.png --platform ios
4

Add device frames to your screenshots:

screenforge frame screenshot.png --device iphone-15-pro
5

Generate ASO-optimized metadata with AI:

screenforge meta generate -n "MyApp" -d "A great app"
